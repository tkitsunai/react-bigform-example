# React + TypeScript + Vite

## Big Form Pattern

画面内でたくさんの入力項目を備えている業務用アプリ等でありがちな巨大フォームの設計パターンを整理する

- 登録するための入力フォームがあり、項目がたくさんある
- 画面に検索機能があり、登録フォームと兼ねている（検索結果を入力項目に反映させる）

### 達成したいこと

- UIとロジックの分離
- 関心分離（usecaseレイヤリング）
- カスタムフックを使用したState管理
- 複雑性を下げるためにuseContext()機能を利用しない
- 巨大なデータ構造のState管理を楽にする
- 入力項目自体を構造管理化に置く
